# LinkVault

LinkVault is a simple front-end project for saving, organizing, and searching important links in one place.

## Features

- Save links with a title, URL, and tags
- Search links instantly
- Filter by favorites and tags
- Store data in the browser using `localStorage`
- Clean landing page with a dashboard workflow

## Project Structure

- `index.html` - page structure
- `styles.css` - visual styling and layout
- `script.js` - app behavior and link management
- `img3.avif` - background image used on the landing page

## How to Run

1. Open the `LinkVault` folder in VS Code or any editor.
2. Open `index.html` in a browser.
3. Use the dashboard to add and manage links.

## Notes

- This project runs entirely in the browser.
- Saved links stay in your browser until you clear site data or `localStorage`.
